## Project Description

This work refers to the entire data analytics work and involves performing data cleaning, data extraction, and data visualization. Major tools used for data cleaning, such as Power Query and Excel, were used here. Power Query was used for past data cleaning, because historical data was used for similarity trend analysis.

It is a collaborative academic, team-based work, and this is partial data analytics work only presented here, because our team built a nutrition-based web application using HTML, CSS, Tailwind CSS, Node.js, and JavaScript.

Worked with a full-stack development team and their web application, the main motive to build this web application for food-based recommendation system, such as BMI calculation, food recommendation for each person, depending on their weight, height, and Age. It provides a people-based diet chart after checking their fitness level.

Gathered past real-time data through online resources and implemented analytical thinking in this concept.

Performed similarity through current data analysis and used India’s food impact analysis in 2024 data, and Global workout trend analysis, which also used 2024 sample data, while comparing current web application analytical thinking was implemented because it improved client-based recommendation by 25 % and fulfilled client requirements when analyzing past data.

The NLP concept is used for text data cleaning, and two main tasks were performed that was sentiment analysis Wheater positive or negative, and most relevant conversation through word counting algorithms are used for entire text data cleaning and many algorithms were used for entire text data cleaning, such as tokenization, word counting algorithm, stop word removal, rule-based sentiment analysis, text preprocessing, and negation handling.

Through this way can identify client needs and can give client-based food suggestions and guidance.

For current data retrieval, Python was used with mysql.connector, tkinter, and pandas.

Here, used mysql.connector to fetch data from the database, then tkinter is used for data storing purposes, while using the filedialog method, and pandas is used for table creation while saving a particular file format.

In the current web application, retrieved details are login information, BMI calculations, and text messages.

These are used MySQL for storing data and Prompt Engineering was used for entire NLP task and this NLP knowledge gained from ISDC college collaborated AI related class and implemented theory knowledge in practically using prompt technic and here faced many difficulty while creating prompt because I received wrong codes while prompt and couldn’t get as I expected result while work so spent more time to get right result, because NLP is huge concept so need more time to learn entire NLP concept.

After facing all the difficulties built an interactive and meaningful report using Power BI and created similarity wise dash board with the current 2025 data.
